SUMMARY
-------

此模組將會在 Views 3.x 模組中新增與校務行政帳號有關的欄位，以方便使用者在自訂頁面與區塊中，顯示原本附加於 token API 裡
的使用者資訊。

目前已提供的欄位包含：
[users][realname]使用者的真實姓名。
[users][userclass]使用者的身份，teacher 或是 student。
[users][birthdate]使用者的生日。
[users][depname]使用者的所屬部門或就讀班級。
[users][titlename]使用者的職稱或座號。
